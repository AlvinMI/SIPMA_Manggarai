# SIPMA (Sistem Prediksi Tinggi Muka Air)

SIPMA adalah sistem berbasis *Machine Learning* untuk memprediksi Tinggi Muka Air (TMA) di Pintu Air Manggarai guna mendukung mitigasi banjir di DKI Jakarta. 

## Fitur Utama
- **Multi-Model Forecasting**: Menggunakan perbandingan model *Stacked LSTM* dan *Random Forest*.
- **Direct Multi-step**: Prediksi 6 jam ke depan secara simultan untuk akurasi yang lebih baik.
- **Interactive Dashboard**: Antarmuka berbasis *Streamlit* untuk pemantauan *real-time*.

## Teknologi
- **Language**: Python 3.x
- **Framework**: Streamlit
- **Deep Learning**: TensorFlow/Keras
- **Machine Learning**: Scikit-Learn

## Cara Menjalankan
1. Clone repository ini.
2. Install dependencies: `pip install -r requirements.txt`
3. Jalankan aplikasi: `streamlit run app.py`
